Cấu hình

Công thức đo lường

ROAS net = (Conv. Value ÷ Spend) × 70% CTR = Click ÷ Impression × 100 CVR = Install ÷ Click × 100 CPM = Spend ÷ Impression × 1.000 CPI = Spend ÷ Install Payrate = Purchase ÷ Install × 100

Chỉ lấy camp có tag (khóa cứng)

Tool chỉ tải & hiển thị camp có tag này trong tên (vd ptt — nhận cả Ptt/ptt, ở đầu/giữa/cuối tên). Áp cho mọi tab + tổng + cảnh báo + heatmap. Để trống = lấy tất cả. Ô tìm kiếm dùng để tìm tiếp trong tập đã khóa.

Thông báo Discord

Cách 1 — chỉ nhận thông báo (text): Discord → Server Settings → Integrations → Webhooks → New Webhook → chọn kênh → Copy Webhook URL → dán vào ô dưới (hoặc set env DISCORD_WEBHOOK_URL).
Cách 2 — có NÚT bấm action (tắt/giảm/tăng budget ngay trong Discord): tạo Discord App + Bot, set env DISCORD_BOT_TOKEN, DISCORD_CHANNEL_ID, DISCORD_PUBLIC_KEY trên Vercel, và đặt Interactions Endpoint URL = /api/discord-interactions. (Nhờ mình set giúp nếu cần.)
